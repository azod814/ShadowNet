# ShadowNet - Advanced Phishing Framework

## Features
- Dynamic website cloning
- Real-time victim tracking
- Fake browser permission dialog for enhanced social engineering
- Modern dashboard for campaign management

## Installation
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`

## Usage
1. Open your browser and navigate to `http://127.0.0.1:5000`
2. Create a new phishing campaign by entering a target URL
3. Share the generated phishing link with your target
4. Monitor victim activity in the dashboard
